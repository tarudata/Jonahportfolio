Cluster Analysis: Policies and demographics

The aim of this project was determine the attitudes of university students in response to the Government’s implementations of Covid-19 restrictions and rules. In this study we collected data from a survey that me and my peers had shared amongst the university’s 1000+ student body Utilizing tools such as R Studio and Excel and methods including factor and clustering analysis to define the various groups from our survey. The result of our analysis lead us to define 2 main groups ‘Health conscious’ and “Risk takers”.

Web Scraping: Getting a bargain

In this project the aim was to build a price tracker to create a spreadsheet which collects variations in price over time in order to both prices of an item a user wishes to purchase.
